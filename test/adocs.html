Integrate secure user authentication into your subdomain in under 5 minutes.

1. Initial Setup

First, create an account in the Developer Dashboard and generate an API key. Ensure your domain is a subdomain (e.g., auth.yoursite.com).

2. Integrate SDK

Add the following script to your website's header:

// Initialize the SDK
KnoxAuth.init('YOUR_API_KEY');

// Trigger Login
function loginNow() {
  KnoxAuth.openLogin();
}

3. Authentication Flow

After the user logs in, they will be redirected to your verification endpoint with the following parameters:

  • r: The User's Email ID
  • s: Your API Secret Key
  • t: A unique session token
